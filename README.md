Projeto Multicol — TCC
======================

Resumo
------
Implementação em Python do jogo Multicol com ênfase na aplicação de lógica proposicional e teoria dos grafos para representar, detectar e pintar faces em um grafo embutido no plano. Este repositório acompanha o Trabalho de Conclusão de Curso (TCC) "Aplicação da lógica proposicional e da teoria dos grafos na implementação digital do jogo Multicol", de Arnaldo Lucas (luc-gh).

Como executar
--------------
Pré-requisitos: Python 3.10+ e a biblioteca pygame.

No PowerShell (a partir da raiz do repositório):

```
python -m pip install --user -r requirements.txt   # se existir; ou: python -m pip install --user pygame
python -m main.src.main
```

Estrutura principal
-------------------
- `main/src/` — código-fonte do jogo (detecção de faces, grafo, renderização, menu, etc.)
- `main/assets/` — imagens e texturas usadas pelo jogo
